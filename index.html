TODO — Cloud integration

This button currently generates the backup JSON locally and simulates the hand-off. To wire up real Google Drive backups:

  1. Register the app in Google Cloud Console and enable the Drive API.
  2. Implement Google OAuth 2.0 (Sign-In) to obtain a user access token — see cloudBackup.authenticate() in script.js.
  3. Use the Drive API files.create (multipart upload) to push the JSON blob produced by Backup.buildExportPayload().
  4. Store the returned file ID / timestamp in settings.cloudBackup for status display.

All hooks are isolated in the CloudBackup module so the rest of the app never needs to change.
